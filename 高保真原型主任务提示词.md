# 三、三端入口设计

请在页面右上角设计一个「端切换器」。

端切换器选项：

1. APP用户侧
2. 租户后台侧
3. 平台管理侧

要求：

- 默认进入「租户后台侧」。
- 端切换器位于顶部 Header 右上角。
- 当前端必须有明显标识。
- 切换端后，左侧菜单立即变化。
- 切换端后，如果当前页面不属于该端，自动跳转到该端首页。
- 三端共用 mock 数据，但必须按端、角色、租户、商家、门店、用户身份进行数据过滤。
- 不同端不能看到不属于自己的功能和数据。

# 五、核心业务流程必须体现

原型中必须体现以下闭环。

## 1. AI问答闭环

用户提问
→ 识别身份
→ 识别租户/商家/门店/渠道
→ 意图识别
→ FAQ / RAG / 工具调用
→ 风控审核
→ AI回复
→ 用户评价
→ 日志记录

## 2. 转人工闭环
